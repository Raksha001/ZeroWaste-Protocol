Rules
Each project is scored by two panels in parallel: AI agent judges (quantitative) and human judges (qualitative). Scores are weighted and combined to determine the final ranking.

Panels	Evaluation methods
Agent judges	OKX Agents automatically scan submitted GitHub repos and onchain transaction history to assess code structure, integration depth with Onchain OS/Uniswap skills, and participation in the X Layer’s ecosystem. Results are compiled into a quantitative report used as the scoring basis.
Human judges	OKX’s internal technical experts and external industry judges review and score projects based on the agent judges’ scores.
Scoring weights	Criteria
Onchain OS/Uniswap integration and innovation (25%)	How creatively and effectively the project uses Onchain OS skills or Uniswap skills.
X Layer ecosystem integration (25%)	How well the project integrates with the X Layer ecosystem and addresses real use cases.
AI interactive experience (25%)	How well the project uses AI skills to create an intuitive user experience.
Product completeness (25%)	The performance, core flow fluidity, and practicality of the product.
